ENG - Expense Tracker CLI

Expense Tracker CLI is a simple command-line application developed in Python for managing personal expenses. The main goal of this project is to allow users to track, update, delete, and summarize 
expenses directly from the terminal.

Expense data is persistently stored in a CSV file called expenses.csv, ensuring that data is preserved between program executions.

The project is built using Python 3 and only standard libraries such as argparse, datetime, and csv, making it easy to run and ideal for learning purposes.

The application can be used by passing arguments directly through the command line. Users can:

Add new expenses with description, amount, and category.

Update existing expenses by ID.

Delete expenses by ID.

List all recorded expenses in a formatted table.

View summaries of total expenses or filter summaries by month.

Each expense has a unique identifier (ID), a description, a category, a recorded amount, and a timestamp for the date of creation.

This project focuses on basic Python concepts such as file handling, data structures, input validation, and command-line application development, making it suitable for beginner, junior, or 
internship-level developers.

PT-BR - Expense Tracker CLI

O Expense Tracker CLI é uma aplicação simples de linha de comando desenvolvida em Python para gerenciar despesas pessoais. O objetivo principal do projeto é permitir que o usuário registre, 
atualize, exclua e resuma suas despesas diretamente pelo terminal.

Os dados das despesas são armazenados de forma persistente em um arquivo CSV chamado expenses.csv, garantindo que as informações não sejam perdidas ao encerrar o programa.

O projeto foi desenvolvido utilizando Python 3 e apenas bibliotecas padrão, como argparse, datetime e csv, tornando-o fácil de executar e ideal para fins de aprendizado.

A aplicação é utilizada por meio de argumentos passados diretamente pela linha de comando. O usuário pode:

Adicionar novas despesas com descrição, valor e categoria.

Atualizar despesas existentes pelo ID.

Excluir despesas pelo ID.

Listar todas as despesas registradas em uma tabela formatada.

Visualizar resumo do total de despesas ou filtrar por mês.

Cada despesa possui um identificador único (ID), descrição, categoria, valor e data de registro.

Este projeto foi criado com foco em aprendizado e prática de conceitos básicos de Python, como manipulação de arquivos, estruturas de dados, validação de entradas e desenvolvimento de aplicações de 
linha de comando, sendo adequado para desenvolvedores iniciantes, juniores ou em estágio.
