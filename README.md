# Expense Tracker CLI

Command-line expense tracker developed in Python for personal finance management and expense monitoring.

## Features

- Add expenses
- Update expenses
- Delete expenses
- List all expenses
- Monthly expense summaries
- CSV data persistence
- Expense categorization
- Command-line interface with argparse

## Technologies

- Python
- Argparse
- CSV
- Datetime
